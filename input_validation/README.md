# Input Validation Module

A Java-based login application with secure authentication using test credentials.

## Overview

This login module provides a simple authentication system for users with predefined roles (scientist, engineer, security).

## How to Run

### 1. Compile

```bash
javac LoginApp.java Main.java
```

### 2. Run

```bash
java Main
```

### 3. Test Credentials

Use the following credentials to test the application:

Username  | Password
scientist | SecurePass123!
engineer  | EngineerKey456@
security  | SecAware789#

To log out for any user, enter 1 and press 'Enter'.
